# Memes

Сюда складывай мемы и исходные сообщения для будущих media-реакций.

Правило на сейчас:
- можно кидать файлы в любом понятном виде;
- позже я смогу переименовать их подряд в числовой формат;
- потом по твоим комментариям занесу описания и теги.

Текущее состояние:
- основной structured catalog лежит в `assets/memes/catalog.json`;
- ручные заметки по смыслу и безопасным категориям лежат в `assets/memes/ROUTING.md`;
- для массовой синхронизации каталога в registry есть `/bot-media sync-pack`.
