# Meme Routing Notes

Нейтральная карта использования картинок без токсичных формулировок.

## Общие правила

- Автоиспользование только при высокой уверенности.
- Использовать редко: только когда картинка выражает реакцию лучше короткого текста.
- Один ответ — максимум одна картинка.
- Один и тот же файл не использовать повторно в тот же день без явной просьбы.
- Персональные или внутренние мемы не отправлять автоматически: только вручную.

## Маршрутизация

- `1` — сильное неодобрение, facepalm, жёсткий негативный вайб.
- `2` — непонимание, странная формулировка, мутный запрос.
- `3` — альтернативная реакция на непонимание, странность, неясный заход.
- `4` — удивление с разочарованием.
- `5` — тёплая похвала, приятное одобрение.
- `6` — успокоить чат, мягко сбить напряжение.
- `7` — кто-то чем-то похвастался, acknowledgement на успех или красивый момент.
- `8` — испуг, перегруз, замешательство, слишком много всего сразу.
- `9` — политический или идеологический спор, резкая тематическая реакция.
- `10` — шок, удивление, резкая реакция на внезапный вброс.
- `11` — визуальное «фу», брезгливость, ручной use-case.
- `12` — очень сильный facepalm, крайнее неодобрение.
- `13` — абсурдные придирки, бессмысленный докап, спор по ерунде.
- `14` — восхищение, уважение, сильная похвала.
- `15` — пустая болтовня, громкие слова без содержания, использовать часто не надо, но это основной мем под такой вайб.
- `16` — жёсткий стоп-сигнал, серьёзное предупреждение, лучше вручную.
- `17` — резерв, пока без автосценария.
- `18` — очень хайповый момент, резкий всплеск интереса или ажиотажа.
- `19` — кринж, неловкий вайб, awkward-реакция.
- `20` — чистая злость, использовать очень редко.
- `21` — персональный внутренний мем, только ручной запуск.
- `22` — персональный внутренний мем, только ручной запуск.
- `23` — очень сильное удивление.
- `24` — повторные вопросы по кругу, заевший диалог, «мы уже это проходили».
- `25` — что-то милое, доброе, wholesome.
